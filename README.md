# Jupyter Docker Pack

Docker image containing a few kernels preinstalled:

 * C++
 * Clojure
 * Haskell
 * ~~Java~~
 * JavaScript
 * Python 2
 * Python 3
 * ~~Scala~~

(No, it does not contain Spark, Tensorflow and the likes).

You can run it with:

    docker run -ti -p 8888:8888 kubuszok/jupyter-pack
